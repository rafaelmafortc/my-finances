# 💸 MyFinances

This project has been born from a personal need: I wanted to get rid of my unfriendly Excel spreadsheet for tracking my monthly budget. So, I have built **MyFinances**, a simple and elegant platform to easily control my **monthly income and expenses**.


## ✨ Key Features

- ✅ Income & Expense tracking  
- 📂 Categorization of expenses  
- 🌎 Currency conversion (BRL ↔ USD)  
- 📊 Interactive and responsive pie charts  
- 🔒 Firebase Authentication  


## 🧱 Technologies

- **Next.js** – fullstack React framework  
- **ShadCN UI + Tailwind CSS** – clean and modern UI  
- **Firebase** – Firestore + Auth for backend  
- **ECharts** – for chart visualizations  
- **Vercel** – for fast and global deployment
